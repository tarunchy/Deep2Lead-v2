🎮 PathoHunt 3D: Design & Plan

A next-generation immersive 3D drug discovery game for kids and teens.

1. Vision & Story

In the year 2085, invisible monsters known as Pathogens have invaded the world's most vital biomes. Traditional medicines are failing. You are a Molecular Hunter, equipped with a Quantum Lab that can materialize "Lead Molecules" as energy projectiles to defeat these giants by finding their weak spots (Binding Pockets).

🌲 The Bio-Jungle

Fight Viral Proteases amidst glowing flora and giant ferns.

🌋 Molten Peak

Battle Heat-Resistant Bacteria near flowing magma rivers.

🏙️ Neon City

Defend the futuristic metropolis from Antibiotic-Resistant Superbugs.

🏜️ Crystal Desert

Engage ancient fungal pathogens in a landscape of silica and glass.

2. Technical Architecture

Three.js (WebGL) Gemma4 AI AutoDock Vina GLSL Shaders
  • The Arena: A full 3D environment with terrain, skybox, and particle effects (fog, embers, digital rain).
  • The Boss: Real 3D Protein Structures fetched from PDB/AlphaFold, rendered as "Crystal Monsters" using custom shaders.
  • Weaponry: Molecules are rendered as glowing 3D spheres (Spacefill style) that the player "shoots" at the protein.
  • Combat Logic:
    • Attack: AI generates 5-10 candidates based on player's "Seed Spell" (SMILES).
    • Damage: Calculated by Molecular Docking (Binding Energy) + Drug-Likeness (QED).
    • Visual Feedback: Floating damage numbers, screen shakes, and monster reactions.

3. Implementation Roadmap

Phase 1: The Arena (New Route)

Create /pathohunt-3d route and pathohunt_3d.html template. Implement the Three.js scene with a "Jungle" theme as the first level.

Phase 2: Monster Materialization

Integrate NGL.js or custom Three.js PDB loader to render the protein structure at the center of the battlefield. Apply a "Boss Aura" shader.

Phase 3: Tactical HUD

Design a "Teen-focused" UI with Health Bars, Energy Gauges, and a SMILES Terminal that looks like a hacking console.

Phase 4: Battle Loop

Connect the frontend to /api/v2/generate (docking-enabled). When the API returns, animate molecules flying from the player to the boss binding site.

4. Sample Integration Code

Adding the Three.js boilerplate for the battlefield:

// pathohunt_3d.js preview
const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
const renderer = new THREE.WebGLRenderer({ antialias: true });

// Environment: Jungle Ground
const groundGeo = new THREE.PlaneGeometry(100, 100);
const groundMat = new THREE.MeshPhongMaterial({ color: 0x1a2e1a });
const ground = new THREE.Mesh(groundGeo, groundMat);
ground.rotation.x = -Math.PI / 2;
scene.add(ground);

// The Boss (Protein) Placeholder
const bossGroup = new THREE.Group();
scene.add(bossGroup);

function animate() {
    requestAnimationFrame(animate);
    bossGroup.rotation.y += 0.005; // Gentle rotation
    renderer.render(scene, camera);
}
